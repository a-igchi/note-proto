# 画面設計

## 概要

本ドキュメントでは、アプリケーションの画面設計について説明する。

## 画面構成

アプリケーションは以下の画面で構成される。

1. **ノート一覧画面**: ユーザーが作成したノートの一覧を表示する画面。
2. **ノート詳細画面**: ノートの内容を表示し、編集や削除ができる画面。

## ノート一覧画面

この画面ではユーザーが作成したノートを、ノートをノード、ノート間のリンクをエッジとするグラフ構造で表示する。

### 1. グラフ

全画面で表示する。拡大縮小、ドラッグによる移動が可能。ノートが存在しない場合は何も表示しない。

#### 1.1 ノード

ノートをノードとして表示する。ノードをクリックすると、ノート詳細画面に遷移する。

右クリックでコンテキストメニューを表示し、以下の操作が可能。

- **削除**: 確認ダイアログを表示し、確定するとノートを削除する。
- **リンクの追加**: リンク追加パレットを表示する。

#### 1.2 エッジ

ノート間のリンクをエッジとして表示する。

右クリックでコンテキストメニューを表示し、以下の操作が可能。

- **削除**: リンクを削除する。

### 2. 新規作成ボタン

新しいノートを作成するためのボタンを配置する。Material UIのFABのような、右下の丸いボタンとする。クリックすると、ノート詳細画面に遷移する。新規ノートのタイトルは空の状態で作成される。

### 3. リンク追加パレット

ノードの右クリックまたはキーボードショートカット（Cmd+K / Ctrl+K）から表示する。ノードが選択されている状態でのみ使用可能。リンクを追加したいノートを選択するためのパレット。選択したノートと、対象のノートの間にリンクが追加される。

- 初期状態では１行のテキストフィールドのみ表示。
- テキストフィールドにノートのタイトルを入力すると、該当するノートの候補がテキストフィールドの下に表示される。
- 以下の操作でリンク先を選択可能。
  - キーボードの上下キーで候補を選択し、Enterキーで決定。
  - 候補をクリックして選択。

## ノート詳細画面

この画面では、ノートの内容を編集できる。

### 1. エディタ

ノートの内容を編集するためのエディタ。以下のすべての要素はボーダーを設けたりせず、一枚のページとして認識できるようにする。

#### 1.1 タイトル

ノートのタイトルを編集するためのテキストフィールド。

タイトルに利用可能な文字は以下のとおり。

- 英数字、日本語（ひらがな、カタカナ、漢字）
- 記号（`_`, `-`, `#` 等）
- 空白文字（スペース、タブ等）は使用不可

タイトルが必須項目であることを最初から伝えるため、placeholder と aria-label には「タイトル（必須）」と表示し、テキストフィールドの下には常に「タイトルは必須です」のヒントを表示する。タイトルが空の場合はエラー状態となり、保存できない。エラー状態のときは、ヒントの代わりにエラーメッセージを表示する。新規作成時は本文が入力された時点でエラー状態に切り替わる（戻るボタンを押した瞬間に初めてエラーが出る、ということを避ける）。

#### 1.2 本文

Markdownエディタ。Milkdownを利用する。WYSIWYGで、Markdownをリアルタイムにレンダリングしながら編集できる。

### 2. 戻るボタン

ノート一覧画面に戻るためのボタンを配置する。右上に配置。戻る際の挙動は以下のとおり。

**新規作成時:**

| タイトル | 本文       | 挙動                                                                       |
| -------- | ---------- | -------------------------------------------------------------------------- |
| 空       | 空         | ノートを保存せず破棄して戻る                                               |
| 有効     | 空 or あり | ノートを保存して戻る                                                       |
| 空       | あり       | エラーを表示し、「破棄して戻る／編集を続ける」を選べる確認ダイアログを開く |

**既存ノート編集時:**

変更内容を保存して戻る。タイトルが空の場合はエラーを表示し、戻らない。
