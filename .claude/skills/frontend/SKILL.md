# Frontend Guidelines

- Use feature based modular architecture. Organize code by feature/domain rather than by type (e.g. components, hooks, utils).
- First think to use component library which is installed in the package. If there is no suitable component, create a new one.

# Design

- Do not use alert, confirm, prompt. Use modals instead.
